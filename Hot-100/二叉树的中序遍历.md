给定一个二叉树的根节点 `root` ，返回 *它的 **中序** 遍历* 。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/09/15/inorder_1.jpg)

```
输入：root = [1,null,2,3]
输出：[1,3,2]
```

**示例 2：**

```
输入：root = []
输出：[]
```

**示例 3：**

```
输入：root = [1]
输出：[1]
```

 

**提示：**

- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`

---

```csharp
// 递归写法：
// 中序遍历：左 -> 中 -> 右
// 声明函数帮助进行中序遍历
// 左节点 （如果当前节点的左节点不为null,继续递归。否则返回当前节点,再往当前节点的右节点进行递归）

public class Solution {
    public IList<int> InorderTraversal(TreeNode root) {
        List<int> result = new List<int>();

        Helper(root, result);

        return result;
    }

    private void Helper(TreeNode node, List<int> res) {
        if (node == null) return;

        Helper(node.left, res);
        res.Add(node.val);
        Helper(node.right, res);
    }
}
```

```csharp
// 迭代写法：
// 通过使用栈模拟中序遍历
// 迭代当前节点，直到最后节点的左子树为null
// 返回栈顶
public class Solution {
    public IList<int> InorderTraversal(TreeNode root) {
        List<int> result = new List<int>();
        Stack<TreeNode> stack = new Stack<TreeNode>();
        TreeNode curr = root;

        while (curr != null || stack.Count > 0) {
            while (curr != null) {
                stack.Push(curr);
                curr = curr.left;
            }

            curr = stack.Pop();
            result.Add(curr.val);

            curr = curr.right;
        }

        return result;

    }
}
```
